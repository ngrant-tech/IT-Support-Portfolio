# Deleted User Recovery

## Overview
Demonstrated hands-on experience restoring a previously deleted user account through the Microsoft 365 Admin Center. This process involved accessing the deleted users directory, initiating the user restoration workflow, creating a new password for the restored account, and returning the user account to the active directory environment.

This project demonstrates practical experience with Microsoft 365 user recovery, identity lifecycle restoration, credential reconfiguration, and administrative account recovery in a cloud-based identity management environment.

---

## Environment / Tech Stack
- Microsoft 365 Admin Center
- Microsoft Entra ID
- User Account Administration
- Identity and Access Management (IAM)
- User Lifecycle Recovery

---

## User Account Restoration Administration
- Accessed **Deleted Users** within Microsoft 365 Admin Center
- Selected a previously deleted user account
- Initiated the **Restore User** workflow
- Created a new password for the restored account
- Completed the account restoration process
- Validated successful account recovery within the Active Users directory

---

## Key Skills Demonstrated
- Microsoft 365 Administration
- User Account Recovery
- User Lifecycle Restoration
- Identity and Access Management (IAM)
- Credential Reconfiguration
- Administrative Recovery Management
- Account Validation

---

## Key Takeaways
Restoring deleted user accounts is an essential administrative function in identity lifecycle management. Effective account recovery supports business continuity, reduces disruption, and ensures users can regain access to organizational resources while maintaining secure credential control.
